Space QBee - Discord Application

Space QBee Application was created for Cubic Castles players as an unofficial utility bot for the game.

Existing Commands

  • /price <name> I 2 options: <amount>, <level>

    User Application Installation: Yes

    Displays the price of an item, contains price information for the following categories:

    • Stamps (weather)
    • Pets
    • Wearables
    • Cars
    • Some fish
    • Some store items
    • Raw materials from mines

    Option Level:

    - Only searching pets by option name works

    Option Amount:

    - Only searching raw materials by option name works.

     

    Price updates are always performed during the weekend. Prices are being updated by the Price Updaters.

     

  • /section <name>

    User Application Installation: Yes

    Display price items in the section. In the list, colored dots mean green is increasing, red is decreasing and white is a stable price.

     

  • /status

    Shows server status and time for Cubic Castles game (If you want to check the in-game time, you can use the command by typing /time in chat). Colored dots indicate server status: green means the server is up, and red means the server is down. The numbers in parentheses show how many minutes the servers have down when marked with a red dot.

     

  • /craft <item>

    User Application Installation: Yes

    See the recipe for the item from the linked website by ccrecipes.

     

  • /realms I 2 options: <search>, <category>

    User Application Installation: No

    View the list to see the following realm names from users.

     

    Option Category:

    - Only Shop, Parkour, Race Track and Adventure can be selected

     

  • /create name <name> <category>

    User Application Installation: No

    Create a name to add to the list of realms. The names can be found using the realms command.

     

    Option Name:

    - Create an existing realm name in the Cubic Castles game

    Option Category:

    - Only Shop, Parkour, Race Track and Adventure can be selected

     

  • /support

    User Application Installation: Yes

    The information explains how to contact a support admin for the Cubic Castles game.

     

  • /leaderboard (available only on the "Cubic Pioneers" Discord server)

    To check the leaderboard for user points from the guessing game.
